# 网易云音乐webapp

#### 介绍

基于Vue所开发的一款webapp

#### 软件架构

软件架构说明
1.Vue  
2.Vuex
3.Vue-router  
4.Webpack  
5.Axios  
6.Fastclick  
7.Vue-lazy

#### 安装教程

1. $git clone (<https://gitee.com/Big_Smail/webapp_of_netease_cloud_music.git>)
2. $npm install
3. $npm run serve 或 $npm start
